My Agenda

我的自選流程

在 Session Explorer 的卡片上按「加入我的流程」，這裡會依日期與時間排序，並自動檢查時間衝突。可用瀏覽器列印功能輸出 PDF。

Team Agenda Exchange

組員流程匯入 / 跨員彙整

每位組員可在「Session 探索/自選流程」選好自己的流程，輸入 nickname 後匯出 JSON 或 CSV；組長可在這裡匯入多位組員的 JSON，獨立彙整，不會覆蓋自己的流程。
